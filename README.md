# TextGlitcher
A simple react text glitcher component


# Installation

```
npm i text-glitcher
```

# Usage

You can use it as any other react component
```
import Glitcher from 'text-glitcher';

...

<Glitcher
  text={"github"}
  intensity={0.5}
  colorIntensity={0.5}
  classProps={classProps}
/>
```
